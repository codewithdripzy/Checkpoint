# Checkpoint

This company is set to track devices efficiently using P2P location sharing and also P2C location pointerss.

- P2P (Peer2Peer): when two checkpoint users passes by each other an adequate set of information is shared instantaneously

- P2C (Peer2Checkpointer): Checkpointers are planted wide range network devices which can be planted in any part of the world and shares informations like GPS, etc. with any user that passes by the checkpointers

## ADdvantages

- Checkpoint can be used to track stolen devices
- Realtime location sharing and tracking
